
## **Applied Computer Science (Network Security Administration Option), Bachelor of Science, Part-time (867EBSC) - BCIT**
### **Program Overview**
- **Type:** **Cybersecurity & Network Management Specialization**.  
- **Delivery:** **Blended (In-person & Online)**.  
- **Status:** **Program currently unavailable**.  
- **Objective:**  
  - Prepare students for **network infrastructure security, administration, and cyber defense**.  
  - Develop expertise in **firewalls, encryption, penetration testing, and secure networking**.

### **Key Learning Areas**
- **Network Security & Cryptography**  
- **Penetration Testing & Ethical Hacking**  
- **Cloud Security & Virtualization**  
- **Advanced Cyber Threat Analysis**  

### **Core Courses**
- **COMP 7006 - Network Security Administration 1 (3.0 credits)**  
- **COMP 8006 - Network Security Administration 2 (3.0 credits)**  
- **COMP 8506 - Special Topics in Network Design & Implementation (3.0 credits)**  
- **COMP 7003 - Introduction to Network Security (3.0 credits)**  
- **COMP 7035 - Operating Systems (3.0 credits)**  
- **COMP 8042 - Advanced Algorithms & Data Structures (3.0 credits)**  
- **COMP 8082 - Project Management (3.0 credits)**  

### **Specialty Courses (Electives)**
- **COMP 7005 - Computer Networks & Protocols (3.0 credits)**  
- **COMP 7402 - Cryptology & Secure Protocols (3.0 credits)**  
- **COMP 8571 - Database Security & Forensics (3.0 credits)**  

### **Admissions & Requirements**
- **Prerequisites:**  
  - **60.0 credits of post-secondary education in computing**.  
  - **Familiarity with C programming recommended**.  
- **International Students:** **Eligible if studying from within Canada**.
